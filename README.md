## C++ Learning Journey

This repository tracks my process of learning C++ as an Object-Oriented Programming language — fundamentals, syntax, and small practice exercises.

### Structure

- `cpp/` — C++ source files: the main focus of this repo.
- `python/` — Python versions of some of the same exercises. Since I already know Python, writing the equivalent logic there helps me understand what the C++ code is actually doing.

### Notes

- Files are small, standalone practice snippets rather than a single cohesive project.
- Not every `.cpp` file has a Python counterpart (and vice versa) — only ones where a side-by-side comparison was useful.
